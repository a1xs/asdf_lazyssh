# asdf-lazyssh plugin for the  [asdf version manager](https://asdf-vm.com/)

A terminal-based SSH manager inspired by lazydocker and k9s - Written in go 
https://github.com/Adembc/lazyssh


## Install plugin
```bash
asdf plugin add lazyssh https://github.com/a1xs/asdf_lazyssh
```
#  Install app

## Install latest version
```bash
asdf install lazyssh latest
```
## Install specific version

```bash
asdf install lazyssh 0.3.0
```
